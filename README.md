# VoltEdge Electrical Website Template

Premium responsive website template for an electrician or electrical contractor. Built with Vite, React, TypeScript, Tailwind CSS, and lucide-react icons.

## Run Locally

```bash
npm install
npm run dev
```

The dev server runs on `http://localhost:5173` by default, or the next available port if that one is busy.

## Build

```bash
npm run build
```

## Customize

Most replaceable content lives in:

```text
src/data/siteContent.ts
```

Update the brand name, emergency number, address, service areas, service list, pricing cards, testimonials, hours, and contact details there.
