# VoltEdge Electrical Website Template

Premium responsive website template for an electrician or electrical contractor. Built with Vite, React, TypeScript, Tailwind CSS, and lucide-react icons.

The template is designed for premium residential and commercial electrical businesses that need strong trust signals, emergency response messaging, clear service paths, realistic estimate anchors, and a modern conversion-focused hero.

## Run Locally

```bash
npm install
npm run dev
```

The dev server runs on `http://localhost:5173` by default, or the next available port if that one is busy.

## Build

```bash
npm run build
```

## Customize

Most replaceable content lives in:

```text
src/data/siteContent.ts
```

Update the brand name, emergency number, address, service areas, service list, pricing cards, testimonials, hours, and contact details there.

## Production Notes

- Replace placeholder contact details, service areas, prices, and credentials before launch.
- Connect CTAs to a real call tracking, booking, CRM, or estimate workflow.
- Replace remote placeholder photography with brand-owned technician, van, panel, and job-site images.
- Keep pricing as starting anchors unless the business has fixed-rate services.
- Add real license/certification numbers, insurance details, service warranty terms, and local SEO pages before publishing for a client.
